# Chatbot sample conversations

Example 1 — memory of sleep issue

- User: "I haven't slept properly for days."
- Assistant: "I'm sorry to hear that — would you like some quick tips to help with sleep?"

- (later) User: "Why do I feel worse today?"
- Assistant: "It looks like you've mentioned sleep issues recently. Poor sleep can definitely make you feel worse — would you like a few ideas to help tonight?"

Example 2 — casual chat should not trigger analysis

- User: "Hey, how's it going?"
- Assistant: "Hey — nice to hear from you. How can I support you today?"
- User: "Just chilling, watching a show."
- Assistant: "I hear you — tell me more if you'd like, or I can help with coping tips."

Example 3 — gratitude

- User: "Thanks for the help"
- Assistant: "You're welcome — I'm here whenever you need to talk."
